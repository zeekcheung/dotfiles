## ðŸš€ Installation

1. Clone the repository.

   ```bash
   # use https
   git clone https://github.com/zeekcheung/.dotfiles $HOME/.dotfiles

   # or ssh
   git clone git@github.com:zeekcheung/.dotfiles.git $HOME/.dotfiles
   ```

2. Run the setup scripts.

   ```bash
   cd $HOME/.dotfiles

   # Arch
   bash arch_setup.sh

   # Ubuntu
   bash ubuntu_setup.sh
   ```
